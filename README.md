## New Mac OSX Setup

This repo contains all of the basic setup/install procedures I generally do on a new mac;

- zsh
- git 
- ssh key
- nvm / avn

```
git clone https://github.com/jordxn/dotfiles ~/.dotfiles && cd ~/.dotfiles
./zsh-setup.sh
./ssh.sh p@ssw0rd
./git.sh "Jordan Schaan" jordan@example.com
```
